# 🚀 Space Shooter

A simple 2D Space Shooter game built with **Python** and **Pygame**.

## 🎮 Features
- Move the spaceship left and right.
- Shoot enemies using the Space key.
- Random enemy spawning.
- Score tracking.
- Game over when an enemy collides with the player.

## 🛠️ Technologies Used
- Python 3
- Pygame

## ▶️ How to Run

1. Clone the repository:
```bash
git clone https://github.com/anjali-devx/SpaceShooter.git
```

2. Install Pygame:
```bash
pip install pygame
```

3. Run the game:
```bash
python main.py
```

## 🎮 Controls
- ⬅️ Left Arrow – Move Left
- ➡️ Right Arrow – Move Right
- Space – Shoot
